# How to play

install these packages
```
pyscroll
pytmx
pygame
```

run
```
python src
```

Enjoy!

# For Developers

## Enable debug

Go to `src/config/__init__.py` and change `debug` to `True` to enable debug mode

## Shortcuts

Shortcut: One of `. + (debug key)`

| Hotkey | Description                                  |
|--------|----------------------------------------------|
| `O`    | Toggle absolute camera mode (see everything) |
| `B`    | Toggle hitboxes                              |
